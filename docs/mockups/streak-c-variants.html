База у всех одинаковая (как тебе понравилось): пульс-кольца на каждый ответ, тряска и молнии с 7-го уровня. Отличается «фишка» среднего тира (3–6). «Вау» начинается уже с 3. Жми «верно» несколько раз подряд.

Движок: level 0..10, «верно вовремя» → +1, ошибка → 0. Цвет — интерполяция «холодный→горячий» без чистого красного (чтобы не путать с ошибкой). В приложении вешается на тип ответа: fast → нарастание, wrong → сброс. prefers-reduced-motion отключает частицы/тряску, оставляя только смену цвета/свечения.
